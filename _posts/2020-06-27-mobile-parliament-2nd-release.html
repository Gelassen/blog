This week I implemented second version of mobile application that provides access to russian government data such as bills, deputies, hearings.

It includes new features such as:
- search by bills
- voting details
- deputies who spoke on hearings
- stenogramm of hearings

Next steps are to learn more about coroutines lifecycle, cover codebase with tests as there were a new architecture was implemented and gather user feedback.
